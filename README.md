# Scouting App (Tauri + Vanilla JS Frontend)

## Helpful Commands

pnpm tauri dev

pnpm tauri android build

## Reminders

- You must install pnpm to run this

- Please increment the version number in package.json as well as in Cargo.toml and tauri.conf.json in the tauri-src folder if you build and upload to releases. The format is year.release.update.

- feel free to contact me at andrewmartinatm@gmail.com if you have any questions (or contact me over any of my other contact info if you have it)
